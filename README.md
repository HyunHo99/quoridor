## 게임 준비 페이지
- 참여하고 싶은 방을 누르고, 그 방의 비밀번호를 입력하면 게임 준비 창에 들어갑니다.
![4](https://user-images.githubusercontent.com/86706463/127528836-a9ebdc99-7a71-4011-9d80-cfd3b0c30788.JPG)
-방에 참여한 두 사람 모두 '준비'버튼을 누르면, 게임이 시작됩니다.
